# Accessibility Statement

**SliceFocus**
*Last updated: March 24, 2026*

SliceFocus is committed to providing an inclusive experience for all users. We have designed and tested the app to work with assistive technologies across all supported platforms.

---

## Supported Accessibility Features

### Screen Reader Support
- **Full VoiceOver support** (iOS, macOS) — all interactive elements, navigation, and schedule data are labeled and navigable
- **Full TalkBack support** (Android) — complete screen reader coverage across all pages
- **Verified reading order** matches the visual layout on every page
- **Semantic labels** reviewed and improved across all screens for clarity

### Visual Accessibility
- **Dynamic Type / Text Scaling** — all text respects the system font size setting, including large and extra-large sizes
- **Bold Text** — honors the system Bold Text accessibility setting
- **High contrast** — information is never conveyed by color alone; all meaningful UI elements use shape, text, or icons in addition to color
- **Dark mode** — full light and dark theme support to reduce eye strain

### Motor Accessibility
- **Reduce Motion** — the app respects the system Reduce Motion setting, replacing animations with instant transitions
- **Keyboard navigation** — interactive elements are navigable via keyboard on desktop (macOS)

### Pages with Full Accessibility Coverage
- Home page (radial chart + timeline)
- Slice detail screen
- Focus session timer panel
- Account settings
- Session history
- Analytics / Productivity Trends
- Actual vs Planned report
